# Быстрый старт
Для подготовки окружения необходимо выполнить следующие команды:

```cmd
git clone https://github.com/fefu-events/frontend.git
cd frontend
npm install
```

Затем сделайте сборку проекта:

```
yarn build
```

Для запуска необходимо выполнить команду:
```
serve -s dist
```
